Pour démarrer le programme :

- Sur Windows :
	1. Double-cliquez sur le fichier COMPILE.bat pour compiler le programme
	2. Un fichier Traverse.exe va se créer dans le même dossier
	3. Double-cliquez sur l'executable pour démarrer le jeu

- Sur Linux :
	1. Déplacez vous à la racine du dossier du jeu dans le terminal
	2. Tapez la commande suivante dans le terminal pour compiler le programme : gcc -L lib -I include src/main.c -o Traverse -lmingw32 -lSDL2main -lSDL2
	3. Pour démarrer le jeu, tapez ensuite : ./Traverse
